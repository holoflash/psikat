# Track Editor

This feature is still highly under development and the current implementation is very much a sketch of what it should be.

## Controls

- **Name**: set the track name
- **Waveform**: choose from one of the built in waveforms or an empty sample
- **Polyphony**: set the number of voices the track can play at the same time (1-8)
- **Loop**: Enable and choose sample loop mode (this feature is still under development)
- **Envelope**: Enable and edit the volume envelope
- **Pitch**: Coarse and fine pitch adjustment
- **Pitch Envelope**: Enable and adjust the pitch envelope
- **Filter**: Enable and adjust the filter envelope

## More about Waveforms

### Built in waveforms

- Sine
- Square
- Saw
- Triangle
- Noise
- Sample

### Loading a sample

- Double click the waveform to open the native file browser and choose a .wav sample
- Drag and drop a sample onto the track editor to load the sample

### Sample/waveform trimming

- Click and drag the edges of the waveform to change the starting and ending points of the sample
